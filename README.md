

## Installation

the first step is:

```bash

python3 -m venv .venv
```

## only for mac and linux

```bash
source .venv/bin/activate

```

# installation

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

# for running 

```bash
uvicorn main:app --reload --port 8000
```
